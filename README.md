### Installation

1. **Clone and navigate to the project:**
   ```bash
   cd /path/to/chatbot
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Set up environment variables:**
   ```bash
   cp env.example .env
   # Edit .env file and add your OpenAI API key
   ```

4. **Start Application:**
   ```bash
   python start_services.py
   ```
